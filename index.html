Nacimiento de la Web

1989

Tim Berners-Lee propone el concepto de la World Wide Web, basado en el uso de hipertexto y el protocolo HTTP.

Las raíces de la Web

1990

Tim Berners-Lee crea el primer navegador web, llamado WorldWideWeb, y el primer servidor web, llamado CERN httpd.

El navegador que revolucionó la web

1993

Se lanza el navegador Mosaic, el primero en mostrar imágenes junto al texto, y el que populariza la web entre el público general

Surge la World Wide Web

1994

Se funda el World Wide Web Consortium (W3C), una organización que se encarga de establecer los estándares y las recomendaciones para la web.

Surge Amazon

1994

Amazon fue fundada por Jeff Bezos, es una tienda online que utiliza HTML, CSS, JavaScript, PHP y otros lenguajes para ofrecer una experiencia de compra personalizada y segura.

Surge JavaScript y PHP

1995

Se crean los lenguajes de programación JavaScript y PHP, que permiten añadir dinamismo e interactividad a las páginas web .

Se integra el concepto de CSS

1996

Se introduce el concepto de hojas de estilo en cascada (CSS), que permiten separar la presentación del contenido en las páginas web.

Se Crear Netflix

1997

Fue fundada por Reed Hastings y Marc Randolph, es un servicio de streaming de películas y series que utiliza HTML, JavaScript, Python y otros lenguajes y tecnologías para ofrecer un contenido de alta calidad y adaptado a las preferencias y dispositivos de los usuarios.

Revolución de Google Chrome

1998

Se lanza el navegador Google Chrome, que se convierte en el más usado del mundo, y que incorpora el motor de JavaScript V8, que mejora el rendimiento de las aplicaciones web.

Surge la Web 2.0

2000

Se inicia la llamada web 2.0, que se caracteriza por el aumento de la participación de los usuarios, el uso de redes sociales, blogs, wikis y servicios web.

Surge WIKIPEDIA

2001

Su primera edición fue el 15 de enero del 2001, es una enciclopedia online colaborativa que utiliza PHP y MySQL para almacenar y mostrar los artículos, que pueden ser editados por cualquier usuario registrado.

Se crea: Ruby on RailS

2004

Se crea el lenguaje de programación Ruby on Rails, que facilita el desarrollo de aplicaciones web basadas en el patrón Modelo-Vista-Controlador (MVC).

Surge GMAIL

2004

Fue creado el 1 de abril del 2004, fue un proyecto iniciado por el desarrollador de Google, Paul Buchheit.

Surge Facebook

2004

Fue creado por Mark Zuckerberg y sus compañeros, es una red social que utiliza PHP, JavaScript, React y otros lenguajes y frameworks para permitir a los usuarios crear perfiles, compartir contenidos, comunicarse y participar en diversas actividades online.

Surge el término AJAX

2005

Se acuña el término AJAX (Asynchronous JavaScript and XML), que se refiere a una técnica que permite actualizar partes de una página web sin recargarla completamente.

Se lanza el framework jQuery

2006

Se lanza el framework jQuery, que simplifica la manipulación del DOM, los eventos, las animaciones y las peticiones AJAX con JavaScript.

Revolución de Python en desarrollo web

2008

Se crea el lenguaje de programación Python, que se convierte en uno de los más populares y versátiles para el desarrollo web, especialmente con el framework Django.

Surge el Concepto del Diseño Web

2010

Se introduce el concepto de diseño web responsivo, que se refiere a la adaptación de las páginas web al tamaño y la orientación de los diferentes dispositivos.

Se lanza el framework Bootstrap

2011

Se lanza el framework Bootstrap, que ofrece un conjunto de herramientas para crear interfaces web con HTML, CSS y JavaScript.
